你好，我是 **xiaoyu233kao**。

这里按照时间记录日常生活、旅行途中遇见的人与风景，以及按下快门时的心情。

这个网站从头到尾都是 AI 做的，GPT，从代码到排版，再到文案。我想，至少这一部分，还是由我自己来写。

摄影对我意味着什么？大一刚入学时，我去面试校团宣，学长问了我这个问题。当时我想了半天，最后说：当我拍完照，几个月后或者几年后再回头看这些照片，还能回忆起当时发生的事、当时的心情。大概是这个意思。面试没过。

后来我看到一本讲摄影的书，开篇第一章就回答了这个问题。他说，摄影是你按下快门定格了这个瞬间，当别人看到的时候，也能产生和你当时身临其境同样的感悟。我醍醐灌顶，觉得非常有道理，下定决心如果下次别人再问我这个问题，我就这么说，很有排面。

但三年过去了，再没人问过我这个问题。摄影对我到底意味着什么，到现在我还是给不出答案。无论是当时面试时仓促应急的发言，还是书里别人说的名言，好像都不是我心里想要的答案。

我常对别人说，我是个记录生活派的摄影师。什么意思呢？背上大包小包，看攻略找个绝佳机位，等个无敌天气，拍出来精修三天，出个大片——这种事我干不出来。

我真正开始接触比较专业的摄影是在高中的时候。高二体艺节，我和朋友们想拍一个短片，一个朋友拿来了他父亲的老相机：佳能 7D，配 17-40，还是个红圈。他对摄影一窍不通，于是把相机全权交给我使用，我一下就爱上了按快门的感觉。

后来成年生日时，我向父母要钱买了一台佳能 800D。上大学后又自己存钱，先后添置了尼康 FE、尼康 Zf、富士 instax WIDE 400 拍立得。以前没相机的时候特别想要一台相机，有了又想要全画幅微单，有了又想要胶片机、拍立得。现在全都有了，反倒是用手机拍照更多了。

这似乎也印证了“记录生活派摄影师”这句话。玩摄影这么多年，拍到现在，倒觉得画质也没那么重要了，能在那个瞬间留下记忆，比什么都宝贵。现在如果有人问我，这么多年拍了什么大片，我还真没什么能拿得出手的作品。相册里塞满的，是我和朋友的笑脸，和我一路的足迹。

网站使用 Hexo 与 Edinburgh 主题构建，并托管在 GitHub Pages。技术上的部分可以交给 AI，但照片里留下的生活，还是我自己的。
